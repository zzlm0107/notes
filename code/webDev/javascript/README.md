## JS简介
JavaScript 是互联网上最流行的web端浏览器脚本语言,被各大浏览器所支持，并且现在依托node让其也可以进行服务端开发，成为真正意义上的编程语言。

概括性的说，JavaScript 是一种基于原型、多范式、单线程的动态语言，并且支持面向对象、命令式和声明式（如函数式编程）风格。
